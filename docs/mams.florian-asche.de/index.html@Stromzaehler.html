DRS110M Stromzähler

Als Stromzähler kann ich den DRS110M von BGTech empfehlen.

Schnittstelle

Dieser ist nicht sehr teuer, und verfügt über eine RS485 Bus Schnittstelle.

Diese Schnittstelle kann man wiederum mit einem RS485 Comserver per Telnet verfügbar machen.

Protokoll

Im MaMS gibt es bereits ein fertiges Protokoll um die Daten abfragen zu können.

Daten

Der Zähler stellt die folgenden Daten zur Verfügung:

voltage Spannung (V)
power Strom (A)
frequency Frequenz (Hz)
active_power Wirkleistung (W)
reactive_power Blindleistung (VAr)
apparent_power Scheinleistung (VA)
power_factor Leistungsfaktor (-)
total_power_meter_reading Gesamtleistung Zaehlerstand (kWh)
total_power_meter_reading_copy Gesamtleistung Zaehlerstand Kopie (kWh)
energy_in_reverse_direction Energie in Rueckwaertsrichtung (kWh)
energy_in_reverse_direction_copy Energie in Rueckwaertsrichtung Kopie (kWh)
temperature_c Temperatur (C)
