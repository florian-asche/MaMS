Module

Webcam

Das Webcam Module dient zum einbinden einer Webcam/LiveStream um das Messgelände im Blick zu behalten. Es können hierbei Bilder von einer anderen Quelle (Server) eingebunden werden, der Server kann ihnen diese Bilder auch herunterladen und zwischenspeichern oder Sie können einfach ein Bild welches sich auf der Server-Festplatte befindet einbinden.
